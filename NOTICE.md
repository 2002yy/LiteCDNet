# NOTICE

This repository is a public, cleaned-up release of a graduation-project codebase for remote sensing change detection.

It contains:
- project-local code for LiteCDNet, SEIFNet integration, ablation experiments, and repository engineering
- adapted baseline implementations collected for unified comparison under one training framework
- a small amount of documentation and illustrative assets for public reproducibility

It does not include:
- datasets
- trained checkpoints
- large visualization outputs
- thesis drafting materials, defense slides, or personal workspace mirrors

Third-party or paper-derived model files are not presented as brand-new standalone official implementations. They are preserved here as project-internal adapted versions for comparison experiments.

For repository-level provenance, see:
- `docs/attribution.md`

For file-level paper / repository mapping, see:
- `docs/references.md`

If you use a specific baseline model from `src/compare/`, please cite the corresponding upstream paper or official repository in addition to citing this repository.
